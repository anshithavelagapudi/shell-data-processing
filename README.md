## Basic commands to start with in Powershell
- mkdir - create directory
- cd - change directory
- ni - new item
- ls - list directory
- pwd - shows working directory
- rm - deletes a directory with files
- rmdir - deletes a directory with empty files
- cat - displays contents of files
